# LoopZen Conversion Hub

This repository contains the LoopZen website built with Vite, React, TypeScript, Tailwind CSS, and shadcn/ui.

## Local Development

Requirements: Node.js and npm.

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Preview (production build)

```bash
npm run preview
```

## Tech Stack

- Vite
- TypeScript
- React
- shadcn/ui
- Tailwind CSS
